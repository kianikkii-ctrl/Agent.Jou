# Journalistieke RSS-agent (Python)

Dit project is een eenvoudige journalistieke agent gebouwd in Python.  
De agent haalt automatisch nieuws op via een RSS-feed en maakt daaruit een geselecteerd overzicht met journalistieke duiding.

## Wat doet deze agent?
De agent voert een vast stappenplan uit:
1. Haalt artikels op uit een RSS-feed (De Morgen)
2. Leest titels en beschrijvingen uit de XML
3. Selecteert de belangrijkste items
4. Voegt eenvoudige journalistieke relevantie toe
5. Genereert een leesbaar nieuwsoverzicht

## Waarom is dit agentic?
Dit script is meer dan een simpele chatbot:
- Het werkt autonoom volgens een vast stappenplan
- Het verwerkt externe input (RSS)
- Het maakt selecties en beslissingen
